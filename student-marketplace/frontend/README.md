# Student Marketplace Frontend

This is the frontend application for the Student Marketplace, a platform designed for students to buy and sell used goods at the end of the school year. 

## Features

- **Account Creation**: Users can create accounts to manage their listings and purchases.
- **Buying and Selling**: Students can list their used items for sale and browse available listings.
- **Location-Based Filtering**: Users can filter listings based on their location to find nearby items.
- **User Profiles**: Each user has a profile to manage their listings and view their purchase history.
- **Responsive Design**: The application is designed to be mobile-friendly for easy access on various devices.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the frontend directory:
   ```
   cd student-marketplace/frontend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm start
```
This will launch the application in your default web browser.

## Components

- **App**: The main component that sets up routing.
- **Header**: Displays the navigation links.
- **Footer**: Contains copyright and contact information.
- **ListingCard**: Represents individual listings in a card format.
- **FilterTabs**: Allows users to filter listings based on categories and location.
